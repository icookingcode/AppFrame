# CSS
  * CSS规则：选择器，以及一条或多条声明
	* 选择器 { 属性:值 ; 属性:值;}
  * CSS Id 和 Class
```
CSS 中 id 选择器以 "#" 来定义
#para1
{
	text-align:center;
	color:red;
} 
```
```
类选择器以一个点"."号显示
.center {text-align:center;} //所有拥有 center 类的 HTML 元素均为居中。
p.center {text-align:center;} //所有的 p 元素使用 class="center" 让该元素的文本居中
```
  * 多重样式的优先级
    * 内联样式）Inline style > （内部样式）Internal style sheet >（外部样式）External style sheet > 浏览器默认样式
