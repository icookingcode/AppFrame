盒子模型 && 边框

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。

小熊打井

山上的小溪(xi)都断流了,大家都说那是因为伐木工人砍倒了太多的树木。小熊住在山上，想要喝水得走老远的山路,可它又不可能搬到别处去，于是想打一口井。有了井,便不愁没有水喝了。

打井可不是很快就能完成的事儿,小熊想找大伙儿一起挖。

小熊首先找到山羊。山羊说:“我年纪大了，腰都快断了,挖不了。”小熊有点失望,接着去找了梅花鹿(lu)。梅花鹿道:“我刚生了几只小鹿，没时间。”小熊没有办法,又去找小白兔。小白兔道:“我天天到处跑，看见哪里有水便喝一点,不需要井。”小熊又找了小鸟、穿山甲、猕猴等朋友,它们都拒绝了。

虽然没有朋友愿意帮忙，小熊仍然不愿放弃:“我一个人挖吧，虽然慢,但总有挖好的一天!”

独自挖井当然很辛苦,但是小熊不怕。它的手在挖土的过程中磨出了血泡，一碰就疼，但是它坚持了下来,血泡破了，变成了厚厚的茧(jin)。半年之后，井挖成了,小熊喝到了甘甜的井水，欣喜得流下了眼泪。

小熊的朋友们听说后，纷纷跑过来看。它们远远望见井口竖起了一块牌子,便止步了,愤怒地说道:“回去回去，它还要收钱哩!”小鸟反驳道:“你们都没有看清楚上面写的是什么,怎么知道是小熊不让我们打水,要来打水必须交钱呢?”山羊道:“你飞得快，去看看写的什么。”小鸟很快又飞了回来，欣喜地说道:“大家快过去,上面的字是小熊写的，说我们都可以打水,都可以;打水!”

大伙儿忽然都感觉挪(nu6)不动脚，站在一起互相望着,脸一个比一个红。

对齐

  • 元素居中对齐：margin: auto;//要水平居中对齐一个元素
  • 文本居中对齐：text-align: center;//文本在元素内居中对齐
  • 图片居中对齐：display: block;margin: auto;
  • 左右对齐：position: absolute;//使用该属性来对齐元素
  • 左右对齐：float: right;//使用该属性来对齐元素
